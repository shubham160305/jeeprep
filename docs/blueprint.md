# **App Name**: DevPrep Academy

## Core Features:

- Video Tutorials: Host tutorial videos covering various frontend development topics.
- Doubt Solving Bot: AI-powered chatbot to answer student queries related to frontend development. It can guide students by understanding the intent behind their question.
- Mock Test Platform: Provide a platform for students to take mock tests mimicking real-world frontend development challenges.
- Hint System: Offer small or big hints based on student's request (using a tool) if they're stuck on a question.
- Answer Validation: Check if the student's answer is correct.
- Progress Tracking: Track student progress and performance in mock tests and video tutorials.
- User Authentication: Allow users to create accounts and save their progress.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and knowledge.
- Background color: Very light blue (#E8EAF6), almost white, for a clean and focused learning environment.
- Accent color: Soft purple (#8E24AA) for interactive elements and hints.
- Body font: 'PT Sans' a humanist sans-serif with a modern look and a little warmth or personality; suitable for body text
- Headline font: 'Space Grotesk' sans-serif with a computerized, techy, scientific feel; Use for headlines
- Use clear, simple icons for navigation and features, following a consistent style.
- Clean and structured layout with clear sections for tutorials, doubts, and mock tests.
- Subtle animations to provide feedback and enhance the user experience, e.g., transitions between questions, loading indicators.
